## Workflow


Branch naming:    feature/STEMM-XX-short-name
Commit messages:  STEMM-XX short description

Example:
  branch:  feature/STEMM-08-login-screen
  commit:  STEMM-08 add login form with validation

Workflow:
1. Pick a Jira ticket (e.g. STEMM-08)
2. Create a feature branch from dev
3. Commit with STEMM-XX prefix on every commit
4. Open PR into dev when ready
5. Partner reviews and merges
